# Bilgisayarı Otomatik Kapatma Programı
## TR

Girilen süre içerisinde, seçilmiş olan modları gerçekleştiren küçük bir programdır.

### Modlar

* **Kapat :** Bilgisayarı otomatik olarak kapatmaya yarar.

* **Yeniden Başlat :** Bilgisayarı yeniden başlatmaya yarar.

* **Uyku :** Bilgisayarı uyku moduna almaya yarar.

* **Oturum Kapat :** Açık olan oturumu kapatmaya yarar.

* **Oturum Kilitleme :** Açık olan oturumu kilitlemeye yarar.

* **Alarm Modu :** Alarm kurmak için kullanılır.


### Özellikler

* ***Türkçe***, ***İngilizce*** ve ***Almanca*** olmak üzere ***3 farklı dil*** seçeneği mevcuttur.

* 7 farklı alarm sesi mevcuttur.

* Girilen süreleri kaydetme özelliği mevcuttur.

***


## EN

It is a small program that performs the selected modes within the entered time.
### **Modes**

* **Shutdown:** Used to automatically shut down the computer.

* **Restart:** Used to restart the computer.

* **Sleep:** Used to put the computer into sleep mode.

* **Log Off:** Used to log off the current session.

* **Lock Session:** Used to lock the current session.

* **Alarm Mode:** Used to set an alarm.

### **Features**

* Available in 3 languages: ***Turkish***, ***English***, and ***German***.

* Includes 7 different alarm sounds.

* Has the ability to save entered times.

## DE


Es ist ein kleines Programm, das die ausgewählten Modi innerhalb der eingegebenen Zeit ausführt.


### **Modi**

* **Herunterfahren:** Dient zum automatischen Herunterfahren des Computers.

* **Neustart:** Dient zum Neustarten des Computers.

* **Ruhezustand:** Dient zum Versetzen des Computers in den Schlafmodus.

* **Abmelden:** Dient zum Abmelden der aktuellen Sitzung.

* **Sitzung sperren:** Dient zum Sperren der aktuellen Sitzung.

* **Alarmmodus:** Wird zum Einstellen eines Alarms verwendet.



### **Funktionen**

* Verfügbar in 3 Sprachen: ***Türkisch***, ***Englisch*** und ***Deutsch***.

* Enthält 7 verschiedene Alarmtöne.

* Verfügt über die Möglichkeit, eingegebene Zeiten zu speichern.
