Bilgisayarı Otomatik Kapatma Programı


Program sayesinden bilgisayarı otomatik olarak, istediğiniz zaman dilimi içerisinde kapatma, yeniden başlatma, oturum kilitleme, oturumu kapatma, uykuya alma gibi işlemleri yapabilirsiniz. İsterseniz alarmda kurabilirsiniz.

Programda şuanlık 3 dil seçeneği mevcut. Türkçe, İngilizce ve Almanca.

Alarm modu için 7 farklı alarm melodisi mevcut.
